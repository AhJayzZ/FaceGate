# Wireless IoT 

- ## 基本規劃
    - ### 1.使用設備
        - ResberryPi

    - ### 2.人臉辨識
        - OpenCV2(LBPH)
        - Dlib

    - ### 3.Server端
        - Flask API
        - MySQL

    - ### 4.IoT資料
        - 人名
        - 狀態
        - 進入時間
        - 出去時間

    - ### 5.工作分配
        - **人臉辨識Dlib**
            紹元、昱寬
        - **網頁呈現**
            勁杰
        - **後端&資料庫**
            振益 
        
- ## 設計流程
    - (必要) 每個人的人臉圖片當資料集(學號_數字_.jpg)，至少30張
    - (必要) 透過時間戳記(Timestamp)來判斷進出
    - (必要) 辨識間隔時間暫定設定成5秒(可變)
    - (優化) 透過紅外線確定是進入還是出去
    - (優化) 兩個門(進入、出去)確定進入還是出去
    - (優化) TensorRT優化模型網路速度
